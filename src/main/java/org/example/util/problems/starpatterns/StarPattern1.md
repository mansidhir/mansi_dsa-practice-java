Problem 1: [Star pattern 1](https://youtu.be/6irHnysGbSI?list=PL7ersPsTyYt2prN058WfA_j3ElgwD1bht)

```java
class Solution {
void hello(){
    //n= user input
    for(int i=1;i<=n;i++) {
        for (int j=0;j<i;j++) {
            System.out.print("*");
        }
        System.out.println();
    }
    }
}
```
